<slicc-add-menu>

A standalone custom element: the + trigger that lives in the prompt bar. Instead of popping a menu, it grows the composer — the search field slides in next to the +/× icon and matching suggestions stack right above it. It lets you upload from this computer (drag & drop or browse), take a photo / screenshot, and run a typeahead search across files in the virtual filesystem, skills, and past conversations. It adapts to light/dark (auto via prefers-color-scheme, or forced with theme="light|dark"), uses Lucide icons, and emits a slicc-add event when something is picked. Demo data is baked in.

Dark theme

The composer is pinned to the bottom of the stage (as it will be in the app). Click + — the box grows upward, the search field appears next to the ×, and results stack above it. Type to filter, ↑/↓ to navigate, Enter to add, Esc to close. Drag a file onto it to upload.
