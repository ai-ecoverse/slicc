<slicc-add-menu>

A standalone custom element: the + trigger that lives in the prompt bar and opens an "add to prompt" dialog. The dialog lets you upload from this computer (drag & drop or browse), take a photo / screenshot, and run a typeahead search across files in the virtual filesystem, skills, and past conversations. It adapts to light/dark (auto via prefers-color-scheme, or forced with theme="light|dark") and emits a slicc-add event when something is picked. Demo data is baked in.

Dark theme

Click +, then type to filter — or drag a file onto the dialog. ↑/↓ to navigate, Enter to add, Esc to close.
